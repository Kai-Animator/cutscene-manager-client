# Cutscene Manager

Simple cutscene manager tool. 

## Description

Created to help integrate production team with designers making it easy for project management.

## Project Tech

* Typescript
* React
* Tailwind

#### Extra Libraries

* FontAwesome for Icons
* Axios for data fetching
